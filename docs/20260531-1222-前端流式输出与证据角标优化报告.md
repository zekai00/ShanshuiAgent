## 本轮改动

1. 新增 `/api/chat/stream`
   - 后端使用 `StreamingResponse` 返回 NDJSON。
   - 先返回 evidence，再流式返回模型 token delta。

2. 前端改为流式读取
   - `ui/modern/app.js` 使用 `ReadableStream` 逐行读取 `/api/chat/stream`。
   - 回答边生成边渲染，不再等待完整回答。

3. 证据引用改成可点击角标
   - 正文中的 `[1]` 或 `（证据[1]：...页码...chunk_id...）` 会渲染成角标按钮。
   - 点击角标会跳转并高亮右侧证据卡。
   - 右侧证据卡仍展示完整来源、页码、chunk_id 和预览。

4. 健康检查补充模型信息
   - `/health` 现在返回 `answer_model`、`answer_provider`、`trained_researcher_lora_exists` 和检索模型路径。
   - 系统页会显示当前回答模型和本地 LoRA 是否存在。
