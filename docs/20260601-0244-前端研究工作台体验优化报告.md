# 前端研究工作台体验优化报告

生成时间：2026-06-01 02:44 CST

## 优化目标

根据用户提出的 8 个体验方向，将前端从“能对话的页面”进一步调整为更像研究阅读工具的工作台。

## 已完成改动

### 1. 对话页信息层级

- 新增回答状态条，显示“已检索证据数、引用条数、最高相关性、当前阶段”。
- 证据卡默认只显示标题、页码、相关性和操作。
- 证据正文改为点击展开，减少右侧栏信息噪音。
- 回答中引用过的证据会高亮，未引用证据弱化显示。

### 2. PDF 预览体验

- PDF 预览保留上一页/下一页。
- 新增页码输入框，可直接跳页。
- 新增放大、缩小、适应宽度/原始大小切换。
- 引用点击仍会打开对应 PDF 页图。

### 3. 回答可信度提示

- 对话页顶部新增轻量状态条。
- RAG 问题显示检索证据数、引用数和最高 rerank 分。
- 非研究问题显示“未检索文献”。

### 4. 非研究问题体验

- 非研究问题返回时自动收起证据栏。
- 不显示旧证据，状态条标明未检索文献。

### 5. 示例问题分组

示例问题从单行按钮改为分组：

- 技法
- 朝代
- 画家
- 作品

用户第一次进入时更容易理解系统适合问什么。

### 6. 语料库页

- 语料库改为“列表 + 详情”结构。
- 左侧显示筛选后的文献列表。
- 右侧显示选中文献详情：分类、作者、类型、页数、权威等级说明、标签和 PDF 下载。
- 后端 `/api/corpus` 新增 `author`、`source_type`、`pdf_url` 字段。

### 7. 系统状态页

- 系统状态拆成普通运行状态和开发诊断。
- 默认显示回答模型、证据库、路由策略。
- Milvus/重排/LoRA 等工程细节折叠在“开发诊断”里。

### 8. 视觉风格

- 工作区去掉明显的装饰性径向光斑。
- 背景动画从点状光斑改为低透明度线性墨痕。
- 主按钮从双色渐变改为更克制的单色青绿。
- 保留入口页氛围，但进入工作台后整体更安静。
