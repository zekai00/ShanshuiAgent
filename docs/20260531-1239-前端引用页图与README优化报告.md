## 设计说明

这次没有把 PDF 路径直接暴露给前端，而是通过 `source_file` 查询项目文献登记表，再由后端返回文件或渲染页图。这样可以把“用户可点击的引用”和“后端可核验的证据来源”连接起来，同时避免任意文件读取。

回答格式同时从两侧控制：后端 prompt 要求模型不要再输出旧标题；前端渲染器对旧格式做兜底清理。这样即使模型偶尔仍按旧格式输出，用户界面也不会回退到不友好的显示方式。
